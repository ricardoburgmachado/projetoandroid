APLICAÇÃO MÓVEL DE AGENDAMENTO

#DADOS DO PROJETO E OBJETIVO 
A aplicação de agendamento constitui-se de um evento do tipo 
agendamento, acessível somente por usuários-cidadão que tem como perfil 
Agente de Saúde. O evento "agendamento é uma especialização do evento 
original modelado na aplicação móvel de agendamento genérica; porém,  com 
as funcionalidades necessárias para a especialidade em questão. Dessa forma, 
a aplicação visa permitir que o Agente de Saúde agende uma consulta, no 
posto de saúde ao qual ele está vinculado, para o usuário (paciente) que o 
agente está realizando a visita.

#COMO USAR
Para fazer o agendamento da consulta, o Agente de Saúde executa o 
evento de agendamento de consultas para preencher os dados necessários ao 
agendamento (Figura 9  -  Agendamento de Consultas). Assim, para fazer o 
agendamento, o Agente de Saúde irá informar o nome do usuário (paciente) e 
a data/hora da consulta. Para isso, o agente poderá clicar no componente 
Data/Hora para escolher o momento desejado para a consulta do usuário e, 
ainda, poderá consultar quais os dias e horários estão disponíveis para o 
agendamento. Vale ressaltar que são carregadas automaticamente pelo 
sistema as informações do Agente de Saúde responsável pelo agendamento e 
o Posto de Saúde no qual será agendada a consulta. Este recurso está 
disponível devido ao agente (usuário) já ter passado pelo processo de 
autenticação no sistema (o sistema já sabe qual é o agente que está utilizando 
a aplicação) e o agente é vinculado a um determinado Posto de Saúde. Dessa 
forma, o agente só poderá agendar a consulta, utilizando a aplicação 
disponível, no posto de saúde ao qual possui vínculo (característica existente 
atualmente nos Postos de Saúde e descritos nos relatórios anteriores).
Após o agente preencher o formulário com as informações necessárias 
para o agendamento de consultas, o sistema irá mostrar uma mensagem de 
confirmação do agendamento com as respectivas informações da mesma. 
Assim, será visualizada uma mensagem constando o usuário (Paciente) para o 
qual  foi marcada a consulta, a data e hora da mesma, o agente de saúde 
responsável pelo agendamento e, ainda, o posto de saúde aonde a consulta 
será realizada. Está previso que o paciente após processamento e confirmação 
do agendamento pelo posto, receba um “SMS” reforçando a confirmação do 
agendamento com os dados mencionados acima.
